# User Story Template

## User Story

**As a** [role],
**I need** [function],
**So that** [benefit].

## Acceptance Criteria

```gherkin
Given [some context]
When [certain action is taken]
Then [the outcome of action is observed]
```

## Definition of Done

- [ ] All acceptance criteria met
- [ ] Unit tests written and passing
- [ ] Code reviewed and merged
- [ ] Story moved to Done column in Kanban board
